# Employee-Attendance-Tracker
Designed and developed a dynamic Excel-based Attendance Tracker for monitoring employee attendance.

Used Excel formulas, conditional formatting, and data validation to automate daily, monthly, and yearly tracking.

Incorporated features like auto date generation, attendance status (P/A/L), and summary dashboards.

Enabled quick calculation of attendance percentage, leave count, and total working days.

Created a clean, professional layout for HR and management use.

Improved efficiency of attendance reporting and employee monitoring using Excel automation.
